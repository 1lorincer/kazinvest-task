<h1>Chat app</h1>

<h2>Client часть</h2>

<b>Зависимости</b>
```
cd ./client && npm i
```
<b>Локальный запуск</b>
```
npm run dev
```
<b>Сборка</b>
```
npm run build
```

<h2>Server часть</h2>

<b>Зависимости</b>
```
cd ./server && npm i
```
<b>Локальный запуск (dev)</b>
```
npm run dev
```
<b>Prod запуск</b>
```
npm run start
```
